## 使用
本仓库为HotPEBuildTools(HotPE的构建工具)

**适用于Win10/11镜像，11版本需低于等于22000.282**

### 1.开发环境

克隆本仓库，安装Python，然后挂载原版系统镜像到虚拟光驱

### 2.生成HotPE

运行 "\HotPEBuildTools.PY"  根据提示操作

演示视频：https://www.bilibili.com/video/BV1W3411a7YW
